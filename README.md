# QA Automation Portfolio

An end-to-end Quality Assurance project demonstrating manual testing, UI automation, API testing, database validation, performance testing, and CI/CD practices commonly used in professional software development environments.

## Technologies

* Selenium
* Playwright
* Java
* TestNG
* REST Assured
* Postman
* PostgreSQL
* SQL
* Jenkins
* Docker
* JMeter
* Jira
* Git & GitHub
* Allure Reports

## Project Scope

This repository showcases a complete QA workflow, including:

* Manual test planning and test case design
* Defect reporting and bug tracking
* UI automation using Page Object Model
* API testing and automation
* Database validation with SQL
* Performance testing with JMeter
* Automated reporting with Allure
* CI/CD integration through Jenkins
* Containerized execution with Docker

The goal of this project is to simulate the responsibilities of a QA Automation Engineer working within an Agile software development team while demonstrating industry-standard testing tools and practices.
